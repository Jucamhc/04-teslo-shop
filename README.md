# Testlo API

1. Clonar Proyecto
2. ```yarn install```
3. clonar el archivo ```example.env``` y renombrarlo a ```.env```
4. cambiar las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```
6. Levantar: ```yarn start:dev```
